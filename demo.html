JsNotify

Pure javascript port of Notify.js

Demo

Message
""
Text to be displayed
clickToHide
true
Hides pop up on click
autoHide
true
Hides pop up after autoHideDelay elapsed
autoHideDelay
5000
If autoHide is true, delay in milliseconds before pop up is hidden
arrowShow
true
If true, show the arrow pointing at the target element
arrowSize
5
If arrowShow is true, sets the arrow size in pixels
position
"bottom left"

Position of notification around target element.

Possible values are

  • top left
  • top center
  • top right
  • right top
  • right middle
  • right bottom
  • bottom left
  • bottom center
  • bottom right
  • left top
  • left middle
  • left bottom
className
"error"
class name set for pop up element. Pre-defined classes are "error", "success", "warn", "info"
gap
2
padding between target element and notification in pixels

Usage

Load jsnotify css and javascript

<head>
    <link rel="stylesheet" href="path/to/jsnotify.css">
    <script defer src="path/to/jsnotify.js"></script>
</head>

Show a notification

<script>
    jsnotify("This is a notification", document.querySelector("input.target"), {autoHide: false, position: "bottom left"});
</script>

Syntax

jsnotify(<text message>, <target element>, [options]

Where:

text message The text to be displayed in notification
target element The HTML element to serve as reference point for the notification's position
options An object containing options for the notification. See demo for reference
  • Adjust notification position on page/element resize
  • Adjust notification position on scroll
